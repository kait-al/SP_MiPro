# Prebiotic+FMT MiPro Microbiota Analysis

This repository contains the R scripts used for analyzing 16S rRNA gene sequencing data derived from FMT donor stool samples incubated in an in vitro gut model (MiPro) in the presence of the prebiotics inulin or XOS.

### Quick Links
- See the R markdown [here](https://kait-al.github.io/SP_MiPro/index.html) for full microbiota analysis walkthrough and figure generation
- Unfiltered counts table [github_counts.txt](https://github.com/kait-al/SP_MiPro/blob/main/data/github_counts.txt)
- Metadata table [github_metadata.txt](https://github.com/kait-al/SP_MiPro/blob/main/data/github_metadata.txt)

### Raw Data
Download raw fastq files from NCBI Sequence Read Archive, [BioProject ID PRJNA1298345](https://www.ncbi.nlm.nih.gov/bioproject/?term=PRJNA1298345).

### Publication
Al KF, Jia S, Silverman M, Reid G, Burton JP, Parvathy S. 2026. Prebiotic Modulation of FMT Donor Microbiota Enhances MASLD-Relevant Taxa and Functions in an In Vitro Gut Model. Journal of Applied Microbiology. doi: 10.1093/jambio/lxag074.
